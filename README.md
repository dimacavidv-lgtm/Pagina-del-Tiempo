# 🌦️ Web del Tiempo

Una aplicación web **minimalista y funcional** para consultar el estado del clima en tiempo real de cualquier ciudad del mundo. Este proyecto utiliza la API de OpenWeatherMap para obtener datos meteorológicos precisos.

## ✨ Características

* **Búsqueda en tiempo real:** Encuentra cualquier ciudad del mundo al instante.
* **Interfaz Limpia:** Diseño enfocado en la legibilidad con una estructura simple y clara.
* **Datos Detallados:**
    * Temperatura actual en grados Celsius (°C).
    * Descripción del clima (Cielo despejado, nubes, lluvia, etc.).
    * Humedad relativa.
    * Velocidad del viento convertida automáticamente a km/h.
* **Iconos Dinámicos:** Los iconos cambian según el estado del tiempo actual (sol, nubes, lluvia, etc.).
* **Manejo de Errores:** Incluye un mensaje visual si la ciudad no existe o el campo de búsqueda está vacío.

## 🚀 Tecnologías Utilizadas

* **HTML5:** Estructura de la página.
* **CSS3:** Diseño personalizado y alineación de elementos mediante Flexbox.
* **JavaScript (ES6+):** Lógica de peticiones a la API y manipulación del DOM.
* **API:** [OpenWeatherMap API](https://openweathermap.org/api).
